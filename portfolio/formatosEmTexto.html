Shape-Outside

Essa propriedade permite que você defina uma forma para que o texto se envolva e flua ao seu redor.

Junto a esta propriedade temos algumas formas basicas

O elemento que vamos aplicar a forma com a propriedade Shape-Outside

precisa ter a propriedade float
  • inset()
  • circle()
  • ellipse()
  • polygon()

inset()

Pode ser usado para criar um retangulo ou quadrado com um deslocamento opcional para o texto que o envolver
